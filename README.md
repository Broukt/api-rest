## Instalación

Para comenzar:

1. Clona el repositorio: 
```bash
git clone https://github.com/Broukt/api-rest.git
```

2. Navega al directorio del proyecto:
```bash
cd api-rest
```

3. Instala las dependencias: 
```
npm install
```

## Uso

Para ejecutar el proyecto localmente, utiliza el siguiente comando:

```bash
npm start
```

Esto inicializará el projecto en un servidor de desarrollo local y se podrá ocupar la colección de Postman para ver su uso.
